# Reseptitietokanta

Tarkoituksena on laatia reseptitietokanta, josta käyttäjät voivat etsiä ja tallentaa muistiin erilaisia reseptejä. Ylläpito voi lisätä tietokantaan uusia ainesosia ja reseptejä, joita käyttäjät voivat hakea niiden erilaisten attribuuttien perusteella. Käyttäjät tallentavat löytämänsä reseptit keittokirjoihin, joille he voivat antaa yksilölliset nimet ja kuvaukset.

#### Toimintoja:
* Käyttäjä
  * Kirjautuminen
  * Keittokirjan lisäys/muokkaus/poisto
  * Reseptien listaus/haku
  * Reseptien lisääminen keittokirjoihin
  * Reseptien poistaminen keittokirjoista
  
* Ylläpito
  * Kirjautuminen
  * Käyttäjien listaus/hallinnointi
  * Reseptien lisäys/muokkaus/poisto
  * Raaka-aineiden lisäys/muokkaus/poisto
